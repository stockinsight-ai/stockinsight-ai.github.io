割安株の見つけ方：PBRとPERを使いこなす

公開日: 2026年4月6日

📖 約5分で読めます

割安株とは何か

「割安株」とは、企業の実力（資産・利益）に対して、株価が低い状態にある銘柄のことです。

いわば「本来は1,000円の価値があるのに、今は700円で売られている」状態です。このような銘柄を発見し、株価が適正水準に戻るまで保有する——これが割安株投資の基本的な考え方です。

割安かどうかを測る代表的な指標がPBRとPERです。

PBR（株価純資産倍率）とは

PBR（Price Book-value Ratio）は、株価が企業の「純資産」の何倍になっているかを示す指標です。

計算式

PBR = 株価 ÷ 1株あたり純資産（BPS）

または

PBR = 時価総額 ÷ 純資産合計

具体例

純資産が1,000億円の会社の時価総額が800億円であれば：

PBR = 800 ÷ 1,000 = 0.8倍

PBRの読み方

  • PBR 1.0倍以下：株価が純資産を下回っている。理論上は「解散価値以下」で買えている状態
  • PBR 1.0〜2.0倍：概ね適正水準
  • PBR 2.0倍以上：成長期待が株価に織り込まれている

日本企業は長らくPBR 1.0倍割れの銘柄が多く、東証もPBR改善を上場企業に求めています。この流れに乗った銘柄は株価上昇の恩恵を受けやすい傾向があります。

PER（株価収益率）とは

PER（Price Earnings Ratio）は、株価が企業の「利益」の何倍になっているかを示す指標です。

計算式

PER = 株価 ÷ 1株あたり純利益（EPS）

または

PER = 時価総額 ÷ 当期純利益

具体例

1株あたり利益が50円の株が1,000円で取引されている場合：

PER = 1,000 ÷ 50 = 20倍

これは「今の利益水準が続いた場合、20年で投資資金を回収できる」という意味でもあります。

PERの読み方

  • PER 15倍以下：割安圏（日本株の歴史的平均は15〜20倍程度）
  • PER 15〜25倍：概ね適正水準
  • PER 25倍以上：成長期待が反映されている（成長株に多い）

ただし、PERは業種によって大きく異なります。成熟産業では低PERが普通ですが、成長産業では高PERが正当化される場合があります。

PBRとPERの組み合わせ方

PBRとPERはそれぞれ異なる角度から割安性を測ります。

状態 PBR PER 解釈
割安の成長株 低 低〜中 最も魅力的
資産株 低 高 利益は出ていないが資産は豊富
成長株 高 高 将来成長への期待が株価に反映
割高株 高 高 注意が必要

本サービスでは、PBRよりもPEGレシオ（PERを成長率で割った指標）を重視しています。理由は、中長期の1〜3ヶ月という時間軸では、成長率を加味した割安性の方が株価パフォーマンスとの相関が高いためです。

注意点：割安だからといって上がるとは限らない

「割安株＝必ず上がる」わけではありません。

  • バリュートラップ：割安なまま放置される銘柄（業績が低迷し続ける）
  • 業種特性の違い：銀行・不動産はもともとPBRが低い傾向がある
  • 一時的な利益急増：特殊要因でEPSが膨らみ、PERが見かけ上低くなる場合がある

指標は必ず複数を組み合わせ、背景にある業績や事業環境もあわせて確認することが重要です。

まとめ

  • PBR：純資産に対する割安度。1.0倍以下で「解散価値以下」
  • PER：利益に対する割安度。15倍以下が割安の目安（業種による）

この2つを入口として使い、さらにROEやPEGといった指標で絞り込むことで、真の割安成長株を見つけやすくなります。
